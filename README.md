# HEGE UI

ESP32-S3 7-inch Touch LCD control panel for HEGE project.

## Hardware

- Waveshare ESP32-S3-Touch-LCD-7
- LVGL v8.3
- ESP Display Panel

## Development

PlatformIO

## Libraries

- ESP32_Display_Panel
- ESP32_IO_Expander
- esp-lib-utils
- LVGL 8.3.11

INSTALLATION PROCEDURE
1. Clone repository

2. Open with VSCode + PlatformIO

3. Build

4. Upload

No extra libraries are required.
